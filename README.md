# Note

## 项目描述

* **简述**：本项目起源于个人记录学习《操作系统真象还原》的心得，复现书中的简易操作系统。后续将对本系统进行裁剪移植。
* **体系结构**：IA32
* **功能**：实模式进入保护模式、启用分页机制、支持部分内外中断、内存管理、任务切换、文件系统并有基本输入输出驱动支持的系统交互。

## 仓库文件结构

* **code** : 本人手码的操作系统源码(已经过模拟检验)，附有详细注释
* **node** : 《**操作系统真象还原》的学习总结
* **os\_memory :** 该操作系统的内存布局图
* **README.md / SUMMERY.md** : 相关说明文件
